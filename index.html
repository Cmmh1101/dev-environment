# Step Comand/Code/file Comments
1 Initialize your folder as an NPM project $ npm init Open your folder in terminal, add the command, and follow the directions/answer question in terminal
2 Lite-server $ npm install lite-server --save-dev Add the command in your terminal to install lite-server
3 Git ignore file Create a file called: .gitignore Create a file in your project directory named .gitignore (Note: the name starts with a period!)
4 Git ignore file add node_modules to .gitignore using VS Code, add the following text to the .gitignore file: node_modules
5 Start Git $ git add . to stage files git
6 Git commit $ git commit -m "commit message" To do your first commit
7 Git push changes $ git push use git push to push the changes to the linked online repository.

1 Compress images $ npm install --save-dev imagemin-cli@5.1.0
NOTE: Some MacOS students have encountered issues with imagemin-cli not installing its plugins due to issues with global permissions on MacOS. If this happens, try:
sudo npm install imagemin-cli@5.1.0 --save-dev --unsafe-perm=true --allow-root 		in terminal add the command to install
2 Compress images JSON script "imagemin": "imagemin img/* -o dist/img", add JSON script
3 Concatenate minify & uglify: Install Usemin $ npm install --save-dev usemin-cli@0.6.0 add command to your terminal
4 Usemin JSON scripts "usemin": "usemin contactus.html -d dist --htmlmin -o dist/contactus.html && usemin aboutus.html -d dist --htmlmin -o dist/aboutus.html && usemin index.html -d dist --htmlmin -o dist/index.html", add script to JSON ** You may need to change files to accomodate with your current project files following the pattern
5 Add build comments HTML comment before bootstrap link:
build:css css/main.css
Comment after style.css:
endbuild 		add those two comments exactly where indicated. Make sure you add the comments expression do the same for the script tags at the bottom. Every html file should have those comments
6 Automate Dist Folder Cleanup $ npm install --save-dev rimraf@3.0.2 in terminal add the command
7 Dist folder JSON script "clean": "rimraf dist", add the script to JSON as you have done previously, the order does not affect but make sure to add a "," after each except the last one
8 distribution folder script in JSON "build": "npm run clean && npm run imagemin && npm run copyfonts && npm run usemin", Add the script to JSON
9 Build the distribution folder $ npm run build On terminal add the command. ** MAKE SURE that you RUN this command before doing commit so the changes are added to the DISTRIBUTION folder
